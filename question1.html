Meta elemants

Defination:Meta elements lives inside the head element and contains information about the web page.

5 Meta tags and their uses:

  1. viewport

    2. Controls how the webpage is displayed on mobile devices.

  2. name

    3. Specifies a name for the metadata.

  3. author

    4. This define the authore of the web page

  4. description

    5. This contains a descriction of the page. This descriction is commomly used by search engine to understand what the page is about.

  5. keywords

    6. This contains a list of comma-separated words that a user might search on to find the page.
