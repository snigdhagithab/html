Semantic tags and it's 5 example:

Defination:The core characteristic of a semantic element is that it clearly communicated its meaning to both the developer and the browser. These elements clearly define its content.

examples:

  1. Aside

    2. The aside element defines some content aside from the content it is placed in

  2. Details

    3. It specifies the tag for additional details.

  3. Footer

    4. The footer element defines a footer for a document or section.

  4. main

    5. This specifies the main page content and should be unique.

  5. Navigation

    6. The nav element defines a set of navigation links.
